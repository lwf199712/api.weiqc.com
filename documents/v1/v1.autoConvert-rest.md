### 二、统计转化数接口
- 请求方式 POST
- 请求地址 : xxx/v1/auto-convert/convert-link/convert
- 请求参数：

名称 | 类型 | 是否必须 | 示例值 | 描述
---|---|---|--- |---
department|string|是|iryClub|微信公众号分部
fansCount|int|是|50|粉丝序号
```json
{
  "department": "iryClub",
  "fansCount": 50
}
```

- 响应的参数如下：

```json
{
    "success": true,
    "data": {
        "status": true,
        "message": "操作成功!",
        "code": 200,
        "data": null
    }
}
```
